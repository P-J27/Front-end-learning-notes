计算时间差，返回相差的天、时、分、秒
需要解决的问题：
1. 两个时间如何相减
//d1是当前时间
var d1 = new Date();
//d2是2019年10月1日，判断距离国庆节放假还有多久
var d2 = new Date(2019, 9, 1);
//在内部调用了d2.valueOf() - d1.valueOf()
console.log(d2 - d1); //返回相差的秒数

2. 函数如何返回多个值
//返回一个对象，让对象的属性返回多个值
function fn() {
return {
year: 1,
month: 2,
day: 3,
hours: 4,
minute: 5,
second: 6
};
}
3. 何如计算相差的天、时、分、秒
8844409202毫秒数
相差的秒数：8844409202 / 1000 % 60

相差的分钟数：8844409202 / 1000 / 60 % 60
相差的小时数：8844409202 / 1000 / 60 / 60 % 24
相差的天数：8844409202 / 1000 / 60 / 60 / 24
